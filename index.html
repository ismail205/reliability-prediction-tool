Formula Used

Reliability

R(t) = e^(-λt)

Reliability is the probability that one asset survives to time t without failure.

Expected Surviving Assets

Survivors = N × R(t)

The expected number of surviving assets equals the asset population multiplied by reliability.

Expected Failed Assets

Failed assets = N × (1 - R(t))

This represents the expected number of assets that experience at least one failure by time t.

Model Assumptions

  • The simulator assumes an exponential failure distribution.
  • Failures are assumed to be random with a constant failure rate.
  • The annual failure rate is assumed to remain unchanged over the full analysis period.
  • All assets are assumed to have the same failure rate and similar operating context.
  • Failures are assumed to be statistically independent.
  • Common-cause failures, systemic failures, ageing, wear-out, infant mortality, and changing operating conditions are not included.
  • The model does not include maintenance interventions, condition monitoring, renewal, replacement, or repair strategy changes.
  • Failed assets means the expected number of assets that experience at least one failure during the period.
  • For repairable assets, this is not the same as total repeat failure events after repair.
  • Results are expected values for education and communication only, not guaranteed outcomes.
